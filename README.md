## Local development

Use the helper script to run the site locally with the Ruby 3.2 compatibility preload already applied.

```bash
chmod +x ./tools/site.sh
./tools/site.sh install
./tools/site.sh serve
```

## Build and test

```bash
./tools/site.sh build
./tools/site.sh doctor
./tools/site.sh test
```
